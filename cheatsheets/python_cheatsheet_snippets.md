# Switch case in python - equivalent

```
>>> def south(): return "South"
... 
>>> def west(): return "west"
... 
>>> switch_case = { 1:south, 2:west}
>>> switch_case[1]
<function south at 0x7eff8812dee0>
>>> switch_case[1]()
'South'
>>> switch_case[2]()
```
